GLASS CLASSIFIER

HOW DOES THIS WORK?

The glass classifier basically takes input of the below mentioned attributes

Random Forest Classification to classify the given glass into one of the seven types of glasses

The attributes used for calculation are:

  • Refractive Index

  • Percentage of Sodium (Na)

  • Percentage of Magnesium (Mg)

  • Percentage of Aluminium (Al)

  • Percentage of Silicon (Si)

  • Percentage of Potassium (K)

  • Percentage of Calcium (Ca)

  • Percentage of Barium (Ba)

  • Percentage of Iron (Fe)

Glasses are classified into the following seven types:

  • Building windows (Float Processed)

  • Building windows (Non Float Processed)

  • Vehicle windows (Float Processed)

  • Vehicle windows (Non Float Processed)

  • Containers

  • Tableware

  • Headlamps
